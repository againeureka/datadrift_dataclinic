## 주요 기능
page 구성은 다음과 같다
- 데이터 로드
- 베이스 시각화
- 임베딩 시각화
- 데이터 드리프트 탐지



## 사용 방법
1. streamlit 접속
    
    ```
    streamlit run main.py
    ```

2. ~~데이터셋 업로드~~(추후 수정 예정)
3. 사이드 바를 통해 각 페이지 확인
4. streamlit 중단

    `ctrl` + `c`
